接口中的常量  public static int a = 1;注意常量必须要初始化并且默认是public static final用来修饰变量，可以不写这个修饰符，程序会自动默认是public static final
接口可以有多个父类
在java8之前，接口中的方法必须是抽象的方法，不能有实现。在java8之后，接口中有2中方法在接口中创建实体方法：1）静态方法,通过“接口名.方法名”调用；2）默认方法，子类可以继承并且重写
在java8之前，接口不能创建对象，但是在8之后可以直接new生成对象，但是必须实现其所有的方法
内部类的本质：
    内部类只是Java编译器的概念，对于Java虚拟机而言，每个内部类都会被编译成一个独立的类，生成独立的字节码文件
    静态内部类,静态内部类可以访问外部类的一切静态资源，但是不可以访问实例变量和方法，内部可以定义静态变量和方法
    成员内部类，可以访问外部类的静态变量和方法也可以访问实例变量和方法，但是不可以定义静态变量和方法
    方法内部类，方法是实例方法可以访问外部类的所有变量和方法，方式是静态方法则只能访问外部类的静态方法和变量。方法内部类可以访问方法中的局部变量，8之前必须加final 修饰符 之后可以不用加但是局部变量不能够赋值
    匿名内部类，与new关联，new后面是父类或者父接口。只能被使用一次
